# 🎓 Academic ER Diagram PDF Viewer

Um projeto de **Visualizador de Diagramas Acadêmicos em PDF**, desenvolvido para facilitar a consulta de modelos conceituais, diagramas entidade-relacionamento (DER) e documentação de sistemas educacionais diretamente no navegador.

Ideal para estudantes, professores, analistas de sistemas e desenvolvedores que trabalham com modelagem de banco de dados e projetos acadêmicos.

---

## 📸 Preview do Projeto

Interface contendo:

- Visualização integrada de arquivos PDF
- Exibição de diagramas entidade-relacionamento
- Barra de ferramentas interativa
- Controle de zoom e navegação
- Download e impressão de documentos
- Área de leitura responsiva

---

## ✨ Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **PDF.js**

## 🚀 Objetivo do Projeto

Fornecer uma solução prática para visualização de documentos PDF contendo modelos conceituais e diagramas acadêmicos, permitindo análise detalhada sem a necessidade de softwares externos.

---

## 📚 Funcionalidades

- Carregamento de documentos PDF
- Navegação entre páginas
- Controle de zoom
- Download do documento
- Impressão direta
- Rolagem horizontal e vertical
- Visualização de diagramas acadêmicos
- Interface responsiva
